# M3Mobile.M3Sdk.Xamarin

This is a Xamarin.Android SDK for M3Mobile devices, providing various functionalities and tools for application development.

NuGet package: [M3Mobile.M3Sdk.Xamarin](https://www.nuget.org/packages/M3Mobile.M3Sdk.Xamarin)

## Requirements

*   **MonoAndroid**: 7.0 or later
*   **Visual Studio**: 2017 or later
